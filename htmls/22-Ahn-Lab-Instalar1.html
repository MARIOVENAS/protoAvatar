Para un mejor desempeño y seguridad el software Ahn Lab sólo opera con:

  • Sistema Operativo Windows 9x / Me / NT Workstation 4.0 2000 / 2003 / Windows XP/Windows Vista/Windows 7 (Windows Vista y Windows 7 son soportados a 32 y 64 Bits).

  • CPU Pentium II recomendado (al menos Pentium 133 MHz) en adelante.

  • Memoria: 128 MB o más, si tiene menos de 128 MB, funcionará pero la página será un poco lenta.

  • Disco Duro: Al menos 100MB de espacio libre.

  • Internet Explorer 7.0 o superior / Firefox 2.0 o superior / Google Chrome / Safari.

Para más información de requerimentos de Software y Hardware por favor visita la sección de "ver más" sobre el software.


Una vez instalado el software este se activará y desactivará automáticamente en cuanto entres y salgas respectivamente, de BancaNet.
