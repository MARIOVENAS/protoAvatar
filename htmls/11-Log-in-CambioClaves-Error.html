CAMBIO DE CLAVES DE ACCESO

Para ingresar a BancaNet y Audiomático es necesario cambiar tusclaves de acceso para realizar tus operaciones bancarias con mayor seguridad.

Para BancaNet

Crea una nueva clave alfanumérica
(8 caracteres)

El primer caracter debe ser una letra

Confirma tu nueva clave alfanumérica
(8 caracteres)

Para Audiomático y/o Banamex Móvil

Crea una nueva clave numérica*
(8 dígitos)

Confirma tu nueva clave numérica
(8 dígitos)

*Si generaste tu número secreto en sucursal y lo cambiaste en Audiomático, genéralo nuevamente.

Clave de acceso alfanumérica:

  • Debe contener tanto números como letras, y ser de 8 posiciones

  • Se permite el uso de letras mayúsculas y minúsculas

  • El primer caracter, debe ser una letra

  • No se permite el uso de letras acentuadas ni caracteres especiales

  • El Número de Cliente no debe estar contenido en el Password

  • No debe contener más de 2 números o letras iguales o consecutivos

  • No debe contener el nombre de la institución (Banamex)

Ejemplo: es diferente "hola1357" que "HOLA1357"

Ejemplo: "ñ","$", comas, etc.

Ejemplo:"12223445", "abbbcd"
