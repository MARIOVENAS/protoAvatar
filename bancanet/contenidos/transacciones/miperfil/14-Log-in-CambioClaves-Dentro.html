Para crear tus claves de acceso utiliza el teclado virtual que se presenta en tu pantalla, este te habilitará las letras o números disponibles para cada casilla en base al cumplimiento de las reglas que abajo se indican.

Posiciona el cursor de tu mouse en cada una de las casillas y selecciona uno a uno el caracter deseado.

  • Debe contener tanto números como letras, y ser de 8 posiciones
  • Se permite el uso de letras mayúsculas y minúsculas
  • El primer caracter, debe ser una letra
  • No se permite el uso de letras acentuadas ni caracteres especiales
  • El Número de Cliente no debe estar contenido en el Password
  • No debe contener más de 2 número o letras iguales o consecutivos
  • No debe contener el nombre de la institución

Ejemplo: es diferente "hola1357" que "HOLA1357"
